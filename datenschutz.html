Datenschutz

1. Datenschutz auf einen Blick

Allgemeine Hinweise

Die folgenden Hinweise geben einen einfachen Überblick darüber, was mit Ihren personenbezogenen Daten passiert, wenn Sie diese Website besuchen. Personenbezogene Daten sind alle Daten, mit denen Sie persönlich identifiziert werden können.

Datenerfassung auf dieser Website

Wer ist verantwortlich für die Datenerfassung?
Die Datenverarbeitung auf dieser Website erfolgt durch den Websitebetreiber (Impressum).

2. Hosting

Wir hosten die Inhalte unserer Website bei folgendem Anbieter:

Supabase

Anbieter ist die Supabase Inc. (im Folgenden Supabase). Wenn Sie unsere Website besuchen, erfasst Supabase verschiedene Logfiles inklusive Ihrer IP-Adressen.

Die Buchungsdaten (Name, E-Mail, Telefonnummer, gewählte Leistungen und Termine) werden in einer von Supabase gehosteten Datenbank gespeichert, um Ihre Terminanfragen bearbeiten zu können.

3. Datenerfassung auf dieser Website

Buchungs- und Kontaktformular

Wenn Sie uns per Buchungsformular Anfragen zukommen lassen, werden Ihre Angaben aus dem Anfrageformular inklusive der von Ihnen dort angegebenen Kontaktdaten zwecks Bearbeitung der Anfrage und für den Fall von Anschlussfragen bei uns gespeichert. Diese Daten geben wir nicht ohne Ihre Einwilligung weiter.

4. Plugins und Tools

EmailJS

Zur Versendung von E-Mail-Benachrichtigungen über Ihre Buchungen nutzen wir den Dienst EmailJS. EmailJS verarbeitet die Formulardaten (z.B. Name, E-Mail-Adresse und Buchungsdetails), um die E-Mail zur Terminbestätigung in unserem Auftrag an Sie zu versenden. EmailJS wird nur geladen, wenn Sie bei den Cookie-Einstellungen der Nutzung von funktionalen Skripten/Cookies zugestimmt haben.

5. Ihre Rechte

Sie haben jederzeit das Recht, unentgeltlich Auskunft über Herkunft, Empfänger und Zweck Ihrer gespeicherten personenbezogenen Daten zu erhalten. Sie haben außerdem ein Recht, die Berichtigung oder Löschung dieser Daten zu verlangen. Wenn Sie eine Einwilligung zur Datenverarbeitung erteilt haben, können Sie diese Einwilligung jederzeit für die Zukunft widerrufen. Hierzu können Sie sich jederzeit an uns wenden.
